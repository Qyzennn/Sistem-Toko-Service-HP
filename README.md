  #  Analisis Pengaruh Frekuensi Login Gagal terhadap Risiko Akun Terkompromi

##  Deskripsi Project
Project ini bertujuan untuk menganalisis hubungan antara frekuensi login gagal dengan risiko akun terkompromi pada sistem bisnis digital.

---

##  Dataset
Dataset yang digunakan:
- authentication_failures_dataset.csv

---

##  Tools
- Python
- Pandas

---

##  Metode
- Statistik deskriptif
- Korelasi sederhana
- Analisis data

---

##  Hasil
- Login gagal berpengaruh terhadap risiko akun
- Semakin tinggi login gagal, semakin tinggi risiko akun
